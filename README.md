I built an aesthetic photo gallery app using SwiftUI that focuses on a clean, modern user experience. I implemented a navigation structure that keeps the main discovery feed and user profile easily accessible, ensuring the flow feels intuitive and familiar. To make the interface more engaging, I added a custom scaling animation to the photo cards using a DragGesture, providing instant tactile feedback when a user interacts with an image. I also utilized @EnvironmentObject to create a global dark mode toggle, allowing the entire app’s theme to switch seamlessly. Between the organized LazyVGrid layout and the interactive sheets for uploading content, I aimed to create a project that balances sophisticated "aesthetic" visuals with practical functionality.
